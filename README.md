# 📘 Estudos de Java para Concursos

Este repositório reúne artigos autorais em **Java** com foco em **Programação Orientada a Objetos (POO)** e análise de questões de concursos públicos, especialmente da área bancária, como o concurso do Banco do Brasil.

O material é produzido como parte de um processo contínuo de estudo, revisão e consolidação de conceitos fundamentais da linguagem.

---

## 🎯 Objetivos

- Revisar fundamentos sólidos de Java  
- Aprofundar conceitos de Programação Orientada a Objetos  
- Analisar questões de concursos comentando detalhes técnicos  
- Explorar comportamentos da linguagem cobrados em prova  
- Produzir material didático estruturado em Markdown e notebooks  

---

## 🧠 Conteúdos Abordados

Os artigos do repositório podem incluir:

- Construtores e blocos de inicialização  
- Ordem de execução em Java  
- Herança e uso de `super`  
- Sobrecarga e sobrescrita  
- Encapsulamento e modificadores de acesso  
- Classes abstratas e interfaces  
- Tratamento de exceções  
- Tipos primitivos e wrappers  
- Análise detalhada de questões de concursos
